### 📊 Regression Analysis on Iris Dataset

### 📖 Project Overview

This project demonstrates Simple Linear Regression using the famous Iris Dataset. The goal was to build a predictive model that estimates Petal Length based on Petal Width and evaluate the model’s performance using standard regression metrics.

### 🎯 Objectives

* Load and explore the Iris dataset.

* Select Petal Width as the independent variable (X).

* Select Petal Length as the target variable (y).

* Split the dataset into training and testing sets.

* Train a Linear Regression model using scikit-learn.

* Make predictions on the test data.

* Evaluate model performance using R² Score and Mean Squared Error (MSE).

* Visualize the regression line and actual data points.

### 📂 Dataset Features

The Iris dataset contains the following measurements:

| Feature      | Description                        |
| ------------ | ---------------------------------- |
| Sepal Length | Length of the sepal in centimeters |
| Sepal Width  | Width of the sepal in centimeters  |
| Petal Length | Length of the petal in centimeters |
| Petal Width  | Width of the petal in centimeters  |

### 📊 Project Workflow

### 1. Data Loading

Loaded the Iris dataset from scikit-learn and converted it into a Pandas DataFrame for easier analysis.

### 2. Feature Selection

Used Petal Width as the predictor variable and Petal Length as the target variable.

### 3. Train-Test Split

Split the dataset into 80% training data and 20% testing data to evaluate the model on unseen observations.

### 4. Model Training

Trained a Linear Regression model to learn the relationship between petal width and petal length.

### 5. Prediction

Generated predictions for the test dataset using the trained model.

### 6. Model Evaluation

Evaluated performance using:

* R² Score – measures how well the model explains the variance in the target variable.

* Mean Squared Error (MSE) – measures the average prediction error.

### 7. Visualization

Plotted the actual data points and the regression line to visualize the relationship between petal width and petal length.

### 📈 Results

The model showed a strong positive relationship between Petal Width and Petal Length, indicating that flowers with wider petals tend to have longer petals.
